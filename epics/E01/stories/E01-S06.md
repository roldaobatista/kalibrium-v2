# E01-S06 — Frontend base (Vite 8 + Tailwind CSS 4 + Alpine.js + Livewire 4)

## Objetivo
Configurar a camada de frontend conforme ADR-0001: Vite 8 como bundler, Tailwind CSS 4 com design tokens base, Alpine.js para interatividade simples e Livewire 4 integrado ao Laravel. O resultado é uma página de layout base renderizável, com hot reload em desenvolvimento e build de produção gerando assets com hash de conteúdo.

## Escopo
- `vite.config.js` configurado com plugin `laravel-vite-plugin` e entrada `resources/js/app.js` + `resources/css/app.css`
- `tailwind.config.js` com `content` apontando para `resources/views/**/*.blade.php` e `app/**/*.php` (para detecção de classes Livewire)
- `resources/css/app.css` com importação do Tailwind CSS 4 (via `@import "tailwindcss"`)
- `resources/js/app.js` importando Alpine.js e Livewire 4 (inicialização via `Livewire.start()`)
- Layout base `resources/views/layouts/app.blade.php` com: `@vite`, meta viewport, `@livewireStyles`, `@livewireScripts`, slot para conteúdo
- Componente Livewire de sanidade `App\Livewire\Ping` com view `resources/views/livewire/ping.blade.php` — exibe `"Livewire OK"` e um botão que incrementa contador (verifica reatividade)
- Rota `GET /ping` (apenas em `APP_ENV != production`) renderizando o componente `Ping`
- `npm run build` gera assets em `public/build/` com hash de conteúdo
- `npm run dev` inicia Vite dev server com hot reload funcional
- PHPStan passando no componente Livewire `Ping`

## Critérios de aceite
- AC-001: `npm run build` retorna exit 0 e gera arquivos em `public/build/manifest.json` com entradas para `app.js` e `app.css`
- AC-002: `php artisan serve` + `curl -s http://localhost:8000/ping | grep -c "Livewire OK"` retorna `1` (componente renderiza no HTML)
- AC-003: `public/build/manifest.json` contém hash de conteúdo nos nomes dos arquivos gerados (verificável com `jq '.[\"resources/js/app.js\"].file' public/build/manifest.json | grep -c "[a-f0-9]\\{8\\}"` retornando `1`)
- AC-004: `./vendor/bin/phpstan analyse app/Livewire/Ping.php --level=8` retorna exit 0
- AC-005: `php artisan livewire:list` lista o componente `ping` sem erros

## Riscos
- Livewire 4 requer configuração específica de CSRF e middleware no Laravel 13 — mitigação: seguir documentação oficial do Livewire 4 para Laravel 13; registrar qualquer ajuste em comentário no `bootstrap/app.php`
- Tailwind CSS 4 mudou a configuração (CSS-first config em vez de `tailwind.config.js`) — mitigação: seguir upgrade guide oficial do Tailwind v4; adaptar content paths conforme novo formato
- Alpine.js e Livewire 4 têm integração nativa (Livewire 4 inclui Alpine internamente por padrão) — mitigação: não importar Alpine separadamente se Livewire já o inclui; verificar documentação antes de duplicar importação
