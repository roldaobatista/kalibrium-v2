# Roadmap — Kalibrium V2

> Lista de épicos por trimestre, em ordem de prioridade. Atualizado conforme o Roldão decide o que vem primeiro.

## Como ler este arquivo

- **Épico** = grande objetivo de produto (ex: "laboratório nunca mais perde prazo de calibração").
- Cada épico vira **histórias** menores em `docs/backlog/historias/`.
- Trimestre dá direção, mas a ordem dentro do trimestre é viva — Roldão pode reordenar a qualquer momento.

## Símbolos

- `[ ]` = não começou
- `[~]` = em andamento (alguma história ativa)
- `[x]` = entregue (todas as histórias do épico aceitas e no servidor)
